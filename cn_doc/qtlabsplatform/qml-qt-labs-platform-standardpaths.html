详细描述

The StandardPaths singleton type provides methods for querying the standard system paths. The standard paths are mostly useful in conjunction with the FileDialog 和 FolderDialog types.


  FileDialog {
      folder: StandardPaths.writableLocation(StandardPaths.DocumentsLocation)
  }

Note: Types in Qt.labs modules are not guaranteed to remain compatible in future versions.

参见 FileDialog, FolderDialog, 和 QStandardPaths.

Markdown.css (WebEngine Markdown Editor example)

Markdown.css is better default styling for your Markdown files.

markdown.css is used in the WebEngine MarkDown Editor example

The sources can be found in tqtc-qt5/qtwebengine/examples/webenginewidgets/markdowneditor/resources/3rdparty/markdown.css.

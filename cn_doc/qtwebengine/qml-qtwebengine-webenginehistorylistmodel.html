WebEngineHistoryListModel QML Type

A data model that represents the history of a web engine page. 更多详情...

Import Statement: import QtWebEngine 1.8
Since: QtWebEngine 1.1
  • 所有成员清单，包括继承而来的成员

详细描述

The WebEngineHistoryListModel type exposes the title, url, icon, and offset roles. The title, url and icon specify the title, URL, and favicon of the visited page. The offset specifies the position of the page in respect to the current page (0). A positive number indicates that the page was visited after the current page, whereas a negative number indicates that the page was visited before the current page.

This type is uncreatable, but it can be accessed by using the WebEngineView.navigationHistory property.

Twitter Timeline Example

Demonstrates authenticating with OAuth to access a Twitter timeline.

The Twitter Timeline example uses OAuth, as supported by Qt网络认证, to sign in to Twitter and display a timeline of tweets (in text format) associated with the authenticated user.

To use this example, a consumer key and secret from Twitter are needed. To register the application visit https://apps.twitter.com. You’ll need to add http://localhost:1337/callback as a callback URL in your Twitter app settings.

运行样例

To run the example from Qt Creator, open the Welcome mode and select the example from 样例. For more information, visit Building and Running an Example.

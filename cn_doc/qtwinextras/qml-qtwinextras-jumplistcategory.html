JumpListCategory QML Type

Represents a jump list category. More...

Import Statement: import QtWinExtras 1.0
Since: QtWinExtras 1.0
  • 所有成员清单，包括继承而来的成员

属性

  • title : string
  • visible : bool

详细描述

The JumpListCategory type represents a category that consists of several Jump List destinations and has a title.

属性文档

items : list<JumpListItem>

A list of items.


title : string

The title of the category.


visible : bool

The visibility of the category.

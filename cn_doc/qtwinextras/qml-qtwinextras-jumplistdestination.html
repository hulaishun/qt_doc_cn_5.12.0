JumpListDestination QML Type

Represents a file that can be edited or viewed by an application. More...

Import Statement: import QtWinExtras 1.0
Since: QtWinExtras 1.0
  • 所有成员清单，包括继承而来的成员

属性

  • filePath : string

详细描述

The JumpListDestination type contains a path to a file that users can select from a Jump List to view or edit it with an application.

属性文档

filePath : string

The path to the file.

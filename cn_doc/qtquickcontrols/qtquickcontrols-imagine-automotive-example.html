Qt Quick Controls 2 - Imagine Style Example: Automotive

An automotive user interface using custom Imagine style assets.

This example demonstrates how custom Imagine style assets can be used to style an automotive user interface.

运行样例

To run the example from Qt Creator, open the Welcome mode and select the example from 样例. For more information, visit Building and Running an Example.

Qt Quick Controls 2 - Imagine Style Example: Music Player

An audio player user interface using custom Imagine style assets.

This example demonstrates how custom Imagine style assets can be used to style a user interface for an audio player application.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from 样例. For more information, visit Building and Running an Example.

Qt WebSockets C++ Classes

List of C++ classes that enable WebSocket-based communication. 更多详情...

This module was introduced in Qt 5.3.

命名空间

QWebSocketProtocol

Contains constants related to the WebSocket standard

类

QMaskGenerator

Abstract base for custom 32-bit mask generators

QWebSocket

Implements a TCP socket that talks the WebSocket protocol

QWebSocketCorsAuthenticator

Authenticator object for Cross Origin Requests (CORS)

QWebSocketServer

Implements a WebSocket-based server

详细描述

To use these classes in your application, use the following include statement:


  #include <QtWebSockets/QtWebSockets>

为了链接依赖该模块，需要增加如下一行指令到你的 qmake .pro 工程文件中：


  QT += websockets

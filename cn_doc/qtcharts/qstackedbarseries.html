详细描述

The QStackedBarSeries class presents a series of data as vertically stacked bars, with one bar per category.

Each bar set added to the series contributes a single segment to each stacked bar.

See the stacked bar chart example to learn how to create a stacked bar chart.

参见 QBarSet, QPercentBarSeries, and QAbstractBarSeries.

Menus Example (ActiveQt)

The Menus example demonstrates the use of QMenuBar 和 QStatusBar in a QMainWindow to implement an in-place active control.

To build the example you must first build the QAxServer library. Then run qmake and your make tool in examples/activeqt/menus.

The demonstration requires your WebBrowser to support ActiveX controls, and scripting to be enabled.

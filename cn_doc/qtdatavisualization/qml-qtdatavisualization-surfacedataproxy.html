SurfaceDataProxy QML Type

The data proxy for a 3D surface graph. More...

Import Statement: import QtDataVisualization 1.2
Since: QtDataVisualization 1.0
Instantiates: QSurfaceDataProxy
继承的父类是：

AbstractDataProxy

继承的子类有：

HeightMapSurfaceDataProxy 和 ItemModelSurfaceDataProxy

  • 所有成员清单，包括继承而来的成员

属性

  • columnCount : int
  • rowCount : int
  • series : Surface3DSeries

详细描述

This type handles surface data items. The data is arranged into rows and columns, and all rows must have the same number of columns.

This type is uncreatable, but contains properties that are exposed via subtypes.

For a more complete description, see QSurfaceDataProxy.

参见 ItemModelSurfaceDataProxy 和 Qt Data Visualization Data Handling.

属性文档

columnCount : int

The number of columns in the data array.


rowCount : int

The number of rows in the data array.


series : Surface3DSeries

The series this proxy is attached to.

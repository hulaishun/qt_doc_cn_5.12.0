AbstractDataProxy QML Type

Base type for all QtDataVisualization data proxies. More...

Import Statement: import QtDataVisualization 1.2
Since: QtDataVisualization 1.0
Instantiates: QAbstractDataProxy
继承的子类有：

BarDataProxy, ScatterDataProxy, and SurfaceDataProxy

  • 所有成员清单，包括继承而来的成员

属性

  • type : AbstractDataProxy.DataType

详细描述

This type is uncreatable, but contains properties that are exposed via the following subtypes: BarDataProxy, ScatterDataProxy, SurfaceDataProxy.

For more information, see Qt Data Visualization Data Handling.

属性文档

type : AbstractDataProxy.DataType

The type of the proxy. One of the QAbstractDataProxy::DataType values.

ScatterDataProxy QML Type

The data proxy for 3D scatter graphs. More...

Import Statement: import QtDataVisualization 1.2
Since: QtDataVisualization 1.0
Instantiates: QScatterDataProxy
继承的父类是：

AbstractDataProxy

继承的子类有：

ItemModelScatterDataProxy

  • 所有成员清单，包括继承而来的成员

属性

  • itemCount : int
  • series : Scatter3DSeries

详细描述

This type handles adding, inserting, changing, and removing data items.

This type is uncreatable, but contains properties that are exposed via subtypes.

See also ItemModelScatterDataProxy and Qt Data Visualization Data Handling.

属性文档

itemCount : int

The number of items in the array.


series : Scatter3DSeries

The series this proxy is attached to.

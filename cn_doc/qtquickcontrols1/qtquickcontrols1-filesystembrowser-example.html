Qt Quick Controls 1 - File System Browser Example

An example for the TreeView control.

This example project demonstrates the usage of TreeView from Qt Quick Controls 1 - a control to display a tree representation of items from a model derived from the QAbstractItemModel class.

The example displays the home path data given by the QFileSystemModel model.

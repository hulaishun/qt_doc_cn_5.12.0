PCRE2 - Stack-less Just-In-Time Compiler

The PCRE library is a set of functions that implement regular expression pattern matching using the same syntax and semantics as Perl 5.

Optionally used in Qt Core (QRegularExpression). Configure with -system-pcre or -no-pcre to avoid.

The sources can be found in tqtc-qt5/qtbase/src/3rdparty/pcre2/src/sljit.

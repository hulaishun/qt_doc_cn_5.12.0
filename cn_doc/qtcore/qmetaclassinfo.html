详细描述

The QMetaClassInfo class provides additional information about a class.

Class information items are simple name--value pairs that are specified using Q_CLASSINFO() in the source code. The information can be retrieved using name() 和 value(). For example:


  class MyClass
  {
      Q_OBJECT
      Q_CLASSINFO("author", "Sabrina Schweinsteiger")
      Q_CLASSINFO("url", "http://doc.moosesoft.co.uk/1.0/")

  public:
      ...
  };

This mechanism is free for you to use in your Qt applications. Qt doesn't use it for any of its classes.

参见 QMetaObject.

DispatchCompute QML Type

FrameGraph node to issue work for the compute shader on GPU. 更多详情...

详细描述

A DispatchCompute allows work to be issued for the compute shader to run on the GPU. The workGroupX, workGroupY 和 workGroupZ properties specify the work group sizes for the compute shader invocation. ComputeCommand components need to be added to entities to instruct Qt3D to select the materials and geometry from the entities for the compute invocation. The work group sizes for the shader invocation will be the maximum of the work group sizes specified in DispatchCompute 和 ComputeCommand.

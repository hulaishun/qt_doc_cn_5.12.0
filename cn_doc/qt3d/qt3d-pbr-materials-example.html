Qt 3D: PBR Materials QML Example

A QML application that demonstrates using PBR materials.

Pbr-Materials demonstrates using the Qt 3D material system.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from 样例. For more information, visit Building and Running an Example.

Qt WebChannel C++ Classes

List of C++ classes that provide the Qt WebChannel functionality. 更多详情...

This module was introduced in Qt 5.4.

类

QWebChannel

Exposes QObjects to remote HTML clients

QWebChannelAbstractTransport

Communication channel between the C++ QWebChannel server and a HTML/JS client

详细描述

To use these classes in your application, use the following include statement:


  #include <QtWebChannel/QtWebChannel>

To link against the module, add this line to your qmake .pro file:


  QT += webchannel

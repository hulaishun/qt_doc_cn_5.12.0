详细描述

The QImageEncoderControl class provides access to the settings of a media service that performs image encoding.

If a QMediaService supports encoding image data it will implement QImageEncoderControl. This control allows to set image encoding settings and provides functions for quering supported image codecs 和 resolutions.

The interface name of QImageEncoderControl is org.qt-project.qt.imageencodercontrol/5.0 as defined in QImageEncoderControl_iid.

参见 QImageEncoderSettings 和 QMediaService::requestControl().

[pure virtual] QList<QSize> QImageEncoderControl::supportedResolutions(const QImageEncoderSettings &settings, bool *continuous = nullptr) const

Returns a list of supported resolutions.

If non null image settings parameter is passed, the returned list is reduced to resolutions supported with partial settings applied. It can be used to query the list of resolutions, supported by specific image codec.

If the encoder supports arbitrary resolutions within the supported resolutions range, *continuous is set to true, otherwise *continuous is set to false.

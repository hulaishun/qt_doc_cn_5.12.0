Obsolete Members for QPaintEngineState

The following members of class QPaintEngineState are obsolete. They are provided to keep old source code working. We strongly advise against using them in new code.

公共函数

(obsolete) QMatrix matrix() const

成员函数文档

QMatrix QPaintEngineState::matrix() const

This function is obsolete. It is provided to keep old source code working. We strongly advise against using it in new code.

Returns the matrix in the current paint engine state.

Note: It is advisable to use transform() instead of this function to preserve the properties of perspective transformations.

This variable should only be used when the state() returns a combination which includes the QPaintEngine::DirtyTransform flag.

这个函数是在Qt 4.2版本中新增引入的。

参见state() and QPaintEngine::updateState().

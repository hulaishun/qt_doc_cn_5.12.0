[静态函数] QObject *QGenericPluginFactory::create(const QString &key, const QString &specification)

Creates the driver specified by key, using the given specification.

Note that the keys are case-insensitive.

参见 keys().

[静态函数] QStringList QGenericPluginFactory::keys()

Returns the list of valid keys, i.e. the available mouse drivers.

参见 create().

Smooth Scaling Algorithm

Normal smoothscale method, based on Imlib2's smoothscale.

Used in Qt Gui (QImage::transformed() functions).

The sources can be found in tqtc-qt5/qtbase/src/gui/painting/qimagescale.cpp.

成员变量文档

QByteArray QVulkanLayer::description

This variable holds the description of the layer.

QByteArray QVulkanLayer::name

This variable holds the name of the layer.

QVersionNumber QVulkanLayer::specVersion

This variable holds the Vulkan version the layer was written against.

uint32_t QVulkanLayer::version

This variable holds the version of the layer. This is an integer, increasing with each backward compatible change.

相关的非成员

uint qHash(const QVulkanLayer &key, uint seed = 0)

Returns the hash value for the key, using seed to seed the calculation.

这个函数是在Qt 5.10版本中新增引入的。

bool operator!=(const QVulkanLayer &lhs, const QVulkanLayer &rhs)

Returns true if Vulkan layers lhs and rhs have different name, version, or spec version.

这个函数是在Qt 5.10版本中新增引入的。

bool operator==(const QVulkanLayer &lhs, const QVulkanLayer &rhs)

Returns true if Vulkan layers lhs and rhs have the same name, version, and spec version.

这个函数是在Qt 5.10版本中新增引入的。

XML Schema

The xml.xsd file provided by W3C describes the XML namespace, in a form suitable for import by other schema documents.

Used for implementing validation of the the XML schema.

The sources can be found in tqtc-qt5/qtxmlpatterns/src/xmlpatterns/schema/schemas/xml.xsd.

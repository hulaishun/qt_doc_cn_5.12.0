Obsolete Members for Gravity

The following members of QML type Gravity are obsolete. They are provided to keep old source code working. We strongly advise against using them in new code.

Properties

  • angle : real
  • magnitude : real

Property Documentation

angle : real

Angle of acceleration.


magnitude : real

Pixels per second that objects will be accelerated by.

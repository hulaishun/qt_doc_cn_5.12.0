Wayland XDG Output Protocol

The XDG Output protocol is an extended way to describe output regions under Wayland

Used in the Qt Wayland platform plugin.

The sources can be found in tqtc-qt5/qtwayland/src/3rdparty/protocol/xdg-output-unstable-v1.xml.
